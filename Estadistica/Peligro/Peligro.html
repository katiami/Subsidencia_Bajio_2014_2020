Afectaciones por subsidencia en toda la zona de estudio

Preprosesamiento de la base de datos

La base de datos se generó a partir de la capa de velocidades de subsidencia, la cual fue reclasificada en 4 clases en función de los siguientes rangos; > 0 cm se le asignó el valor de 5, -1 cm a -2 cm se le asignó el valor de 4, de -2,1 cm a -4 cm se le asignó el valor de 3, -4,1 cm a -6 cm se le asignó el valor de 2 y > 6 cm se le asignó el valor de 1. Posteriormente se convirtió a polígono y se unieron los polígonos que presentaron el mismo gridcode, posteriormente se generó un campo nuevo de tipo “texto” con el nombre “Clasificio”, y se le asignó una clasificación a cada gridcode en función de los rangos de subsidencia asignados previamente ver Tabla 1.

Bajo 4 -1cm – -2cm
Medio 3 -2.1cm – -4cm
Alto 2 -4.1cm – -6cm

Estadisticas por municipio

Preprocesamiento de la base de datos

La capa utilizada en el apartado anterior posteriormente se intersecto con una capa de municipios obtenida del (INEGI, 2020) y posteriormente se exporto con el formato xlsx con el nombre “base_datos_poligon_geovelocity_intersect” Además, se generó otra base de datos utilizando la capa de municipios de (INEGI,2020) se intersecto con el área de estudio y posteriormente se calculó el área considerada para cada municipio.

Creación de la base de datos

## Importamos las bases de datos
Peligro_MUN <- read_xlsx("base_municipio_clip.xlsx", sheet = 1) 
Peligro_MUN_CLASE <- read_xlsx("base_datos_poligon_geovelocity_intersect.xlsx", sheet = 1) 

### Calculamos el porsentaje de áreas afectadas en la zona de estudio 

Peligro_MUN$POR_AREA <- Peligro_MUN$AREA_CON *100/Peligro_MUN$AREA_KM

Creación de la base de datos

#### Crear una nueva base de datos a partir de las anteriores seleccionamos solo los campos necesarios ##

nueva_base_1 <- (Peligro_MUN[,c("NOMGEO", "AREA_KM")])

nueva_base_2 <- (Peligro_MUN_CLASE [,c("NOMGEO", "Peligro", "AREA_KM")])

## Renombramos la columna AREA_KM de la nueva base de datos 2
 colnames(nueva_base_2)[3] <- "AREA_CON"

### Juntamos las bases de datos 

Base_Final<- merge(nueva_base_1, nueva_base_2, by ="NOMGEO")

### Calculamos el porcentaje 

Base_Final$Porcentaje <- Base_Final$AREA_CON *100/Base_Final$AREA_KM

Gráfica 2: Porcentaje de áreas afectada en cada municipio

P_Area_Afectada <- ggplot(Base_Final, aes(NOMGEO, Porcentaje, fill= factor(Peligro, levels = c("Muy alto", "Alto", "Medio", "Bajo", "Nulo"))))
